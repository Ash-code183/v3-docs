## Mainnet Token Sources

| Network   | Where to Buy                           |
|-----------|----------------------------------------|
| Avalanche | Available on most major exchanges      |
| BSC       | Binance, FTX, Kucoin                   |
| Polygon   | Available on most major exchanges      |
| Celo      | Binance, Coinbase, Kucoin              |
| Arbitrum  | Available on most major exchanges or via the canonical bridge |
| Optimism  | Available on most major exchanges or via the canonical bridge |
| Ethereum  | Available on most major exchanges      |
| Moonbeam  | Binance, Coinbase, Kucoin              |

### Steps for Mainnet Tokens:
1. Acquire tokens from the listed exchanges.
2. Seamlessly withdraw these tokens to the relevant network wallet.
3. For an updated list of exchanges supporting each asset, refer to [CoinMarketCap](https://coinmarketcap.com/) or [CoinGecko](https://www.coingecko.com/).
